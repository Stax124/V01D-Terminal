V01D-Terminal is Python powered flexible Windows terminal for everyday use

subprocess : link

prompt-toolkit : link

Terminal settings:

void config -> prints out settings saved in config.yml
void multithreading [true | false] -> use threads for autocomplete menu
void fuzzycomplete [true | false] -> autocomplete based on characters not words
void start -> start new terminal
void update -> updates all python libraries installed
void title [string] -> change title of this terminal (similar to cmd "title [string]")

Hashing:

Type of hashes supported: md5, sha1, sha224, sha256, sha384, sha512

Usage: (hash) [string] or (hash)sum [file]

md5 hello world -> 5eb63bbbe01eeed093cb22bb8f5acdc3
md5sum config.yml -> 1d884456db9bba914f4535bdfda4abdb

Math:

+ -> Add
- -> Subtract
* -> Multiply
/ -> Divide
% -> Remainder
** -> Amplify
// -> Floor divide

gcd -> greatest common divisor: gcd [float|int],[float|int]
lcm -> lowest common multiple: lcm [float|int],[float|int]
rng -> random number generator: rng [int](included),[int](excluded)

PC Parts:

component -> basic info about all major components
motherboard, cpu, gpu, ram -> more detailed info about each component

Alias:

alias - define your own function: alias sayhello @echo Hello!!
delalias - remove alias: delalias [string](name of alias)
aliases - show all user defined aliases

Other:

ping - never ending ping loop
pagefile - show pagefile location and size
read - read specified .txt file
power - change your Windows powerplan
download - dictionary for downloading files (download -list)
open - open file explorer in current directory
plain2string - convert plain text to strings: plain2string mode[line,file] text/[filename]
elevate, admin - grant admin permission for shell
